CineReview

Site de critique de films, projet de la spécialité Symfony. (en cours d'intégration)
Symfony PHP BootStrap phpMyAdmin MySQL

Description

Mise en place de la sécurité avec le bundle LexikJWTAuthenticationBundle, afin de pouvoir s'authentifier et de pouvoir accéder à des fonctionnalités supplémentaires. Backoffice et gestion des favoris, API pour les posters.
