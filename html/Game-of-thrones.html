Trip O'dviser

Exercice de formation pour le choix de la spécialisation où l'on doit créer un mini site web sur le thème de Game of Thrones.
Laravel Blade PHP phpMyAdmin

Description

Reprise de Laravel que l'on avait vu sur le projet précédent, avec la découverte totale et autonome de Blade.


Créer les pages suivantes : accueil, un personnage, lister les maisons, lister les personnages des maisons


Le tout de façon dynamique (contenu, couleurs), en établissant les relations via une BDD.

Technos

Laravel, Blade, PHP, Bootstrap, HTML 5, CSS 3.
