AuxShoes_Backoffice

Exercice de formation sous PHP : création du backoffice d'un site e-commerce de chaussures (projet AuxShoes).
PHP MySQL phpMyAdmin Bootstrap

Description

Créer un backoffice de notre site AuxShoes à partir de l'intégration fournie et de la BDD existante du projet AuxShoes.


Apprentissages

  • Serveur PHP
  • Requêtes SQL
  • MVC
  • PDO
  • Création et gestion d'une BDD relationnelle, jointures
  • MCD
  • MLD
  • SCRUM Agile
  • User Stories
  • Namespaces suivant la norme PSR-4 (autoload)
  • Regex
  • Templates
  • CRUD
  • Dynamisation des pages
  • Factorisation
  • phpMyAdmin
  • Bootstrap
  • Création et gestion d'un backoffice
  • Sessions
  • Sécurité du site par authentification avec l'Access Control List
  • Tokens anti-CSFR
  • Versioning GitHub avec commandes SH

Technos

PHP 7.4, BootStrap, HTML 5, CSS 3, phpMyAdmin, MySQL
