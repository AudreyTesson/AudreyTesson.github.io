AuxShoes

Exercice de formation sous PHP : création d'un site e-commerce de chaussures.
PHP MySQL phpMyAdmin Bootstrap

Description

Créer les pages suivantes : accueil, détails d'un article, catégories, types de produits, marques


Récupération des données depuis l'intégration déjà fournie.


ATTENTION : le backoffice et la gestion de la BDD ont été faites plus tard, sur un autre dépôt git.

Technos
