Pokédex

Exercice de formation sous PHP : création d'un pokédex.
PHP MySQL phpMyAdmin

Description

Créer un Pokédex en utilisant PHP, MySQL, HTML et CSS, Composer et AltoRouter.
Créer la page d'accueil qui liste tous les pokémon de la BDD et une page détail d'un pokemon, comprenant ses stats et son type (couleur en fonction du type).
Enfin, une page qui liste les types, qui redirige vers une page qui liste les pokémon filtrés par le type cliqué précédement.


Apprentissages

  • PHP
  • Requêtes SQL
  • MVC
  • Factorisation
  • Jointures
  • Composer
  • AltoRouter
  • CSS

Technos

PHP 7.4, HTML 5, CSS 3, phpMyAdmin, MySQL
