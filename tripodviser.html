Trip O'dviser

Projet de formation où l'on reprend un projet avec une intégration HTML/CSS présente, afin d'y implémenter des fonctionnalités via JavaScript tout en découvrant le DOM.
JavaScript Bootstrap

Description

Site internet de voyage, où le but est créer des scripts JavaScript et ainsi de rendre le site dynamique.


Il est prévu de permettre de changer de mode de thème (dark / light), de couleur de thème (vert / bleu / rouge)


L'ajout d'un slider de photos fonctionnel et la possiblité d'ajouter (en cache) des favoris.


Création d'une modal newsletter contenant un filtre et messages d'erreurs en cas d'utilisation d'emails jetables.


Fonction de filtre dynamique des reviews via la sélection du nombre d'étoiles.

Technos

JavaScript, Bootstrap 5.1, HTML 5, CSS 3.
